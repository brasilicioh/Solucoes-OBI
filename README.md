# Sobre esse repositório
Esse repositorio contém algumas soluções da Olimpíada Brasileira de Informática (OBI).
É possível realizar códigos com: Python, C++, C, JS e Rust.

Dentro da pasta `./checks` tem pasta de questões, havendo suas respectivas entradas do código e saídas esperadas. Na pasta `./solucoes` há as pastas com respectivas soluções de determinada questão.

Ao topo de cada solução na pasta de soluções, deixar anotado alguma referência da questão, como: `nome questao - OBI ano fX nY` (onde nome é o título da questão da prova, ano é o ano da prova, f é a fase e n é o nível - pode ter a mesma questão em níveis diferentes).

Caso tenha questões de mesmo nome, a partir da segunda, referenciar com algum numeral tanto a pasta dentro das soluções, dentro dos checks e o proprio nome do arquivo da solução.

Como exemplo de uso do repositório, veja `exemplo` (soma de dois inteiros) e `exemplo2` (concatenação de n textos).

# Como usar
Clone este repositório e execute o comando ```python validar.py programa.py``` (onde "programa" é o nome dado da questão).

> O ```programa.py``` deve ter o mesmo nome da pasta que contém as entradas e saídas e a pasta em que ela está armazenada.

# Como contribuir
Você pode contribuir para esse repositório adicionando soluções para problemas incluindo comentários explicando o código ou adicionando as entradas e saídas esperadas do site da OBI.
